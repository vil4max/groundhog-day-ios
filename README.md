# Groundhog Day

Personal countdown app with flip-clock UI and Groundhog Day theme: pick a future date and swipe through years → seconds on split-flap tiles, with optional local notifications.

## Stack

iOS 17+ · SwiftUI · `@Observable` · Swift Concurrency · UserDefaults (+ iCloud KV stub, off by default) · en / ru / uk · `com.vil4max.groundhogday`
